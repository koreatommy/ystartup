# Y-START 서비스 랜딩 페이지 제작 문서

## 1. 문서 개요

본 문서는 **Y-START 창업 교육 서비스**의 랜딩 페이지를 제작하기 위한 기준 문서입니다.  
실제 서비스 구조, 기능, 콘텐츠 자산, UI 흐름을 기준으로 랜딩 페이지의 목적, 정보 구조, 섹션 구성, 콘텐츠 방향, 구현 원칙을 정리합니다.

본 문서는 기획, 디자인, 프론트엔드 개발이 동일한 기준으로 협업할 수 있도록 작성합니다.

---

## 2. 서비스 정의

### 2.1 서비스명
**Y-START**

### 2.2 서비스 한 줄 정의
청소년 창업 교육을 위한 **10단계 디지털 워크북 학습 플랫폼**입니다.

### 2.3 서비스 핵심 설명
Y-START는 아이디어 발굴부터 실행, 운영, 피칭까지 이어지는 창업 교육 과정을 단계별 워크북으로 제공하는 웹 기반 학습 서비스입니다.  
사용자는 각 단계별 워크북을 열람하고, PDF 기반 실습을 수행하며, 보조 학습 도구를 통해 창업 사고력을 확장할 수 있습니다.

### 2.4 핵심 가치
- 창업 여정을 단계별로 구조화하여 학습 부담을 낮춤
- PDF 워크북 기반 실습과 주석 기능 제공
- 5 Whys 기반 보조 활동으로 사고 확장 지원
- 시각 자료와 단계별 콘텐츠를 통해 학습 몰입도 향상

---

## 3. 랜딩 페이지 제작 목적

랜딩 페이지는 단순 소개 페이지가 아니라, 사용자가 서비스의 구조와 가치를 빠르게 이해하고 바로 학습 행동으로 이어지도록 설계해야 합니다.

### 주요 목적
1. 서비스가 무엇인지 10초 내 이해시키기
2. Y-START가 제공하는 학습 방식과 차별점을 명확히 전달하기
3. 사용자가 즉시 1단계 학습 또는 워크북 탐색을 시작하도록 유도하기
4. 창업 교육 프로그램으로서의 체계성과 완성도를 시각적으로 전달하기

---

## 4. 타깃 사용자

### 4.1 1차 타깃
- 청소년 창업 교육 참여자
- 창업 동아리 및 창업 교육 수강생
- 팀 단위 실습을 수행하는 학습자

### 4.2 2차 타깃
- 교사 및 교육 운영자
- 학교·기관 프로그램 관리자
- 교실 또는 교육 현장에서 디지털 워크북을 활용하려는 담당자

### 4.3 사용자 니즈
- 단계별로 정리된 창업 교육 흐름이 필요함
- PDF 기반 워크북을 온라인에서 쉽게 보고 활용하고 싶음
- 학습 내용이 복잡하지 않고 직관적으로 안내되길 원함
- 실제 교육 운영에 적합한 구조와 보조 기능이 필요함

## 5. 랜딩 페이지 핵심 메시지

### 메인 메시지
**꿈을 현실로 만드는 창업 여정**

### 보조 메시지
아이디어에서 실행까지, 청소년 창업 교육의 전 과정을 10단계 워크북으로 체계적으로 안내합니다.

### 전달 포인트
- 창업 교육에 최적화된 디지털 워크북 플랫폼
- 단순 자료 제공이 아니라 단계별 실습 중심 구조
- PDF 학습, 시각 자료, 보조 활동이 연결된 학습 경험

## 7. 섹션별 상세 기획

## 7.1 히어로 섹션

### 목적
서비스의 정체성과 핵심 가치를 첫 화면에서 명확히 전달합니다.

### 포함 요소
- 메인 헤드라인: **꿈을 현실로 만드는 창업 여정**
- 서브 헤드라인: **아이디어에서 실행까지의 로드맵**
- 설명 문구: 청소년 창업 교육을 위한 10단계 디지털 워크북 플랫폼이라는 점을 간결하게 설명
- 대표 CTA 버튼: **1단계부터 시작하기**
- 보조 CTA 버튼: **워크북 둘러보기** 또는 **커리큘럼 보기**
- 시각 요소: 로드맵 이미지, 창업 교육 분위기를 전달하는 그래픽 또는 UI 미리보기

### 기획 의도
히어로는 마케팅 카피보다 교육 서비스의 구조적 신뢰를 우선 전달해야 합니다.  
과장된 문구보다 “무엇을 어떻게 제공하는 서비스인지”가 선명하게 드러나야 합니다.

---

## 7.2 핵심 기능 소개 섹션

### 목적
Y-START가 단순 소개형 사이트가 아니라 실제 학습 도구라는 점을 보여줍니다.

### 권장 카드 구성

#### 1) 10단계 워크북 학습
팀 빌딩부터 피칭까지 이어지는 창업 교육 과정을 단계별 워크북으로 제공합니다.

#### 2) PDF 실습 기능
브라우저 내에서 워크북 PDF를 열람하고, 필기·형광펜·텍스트·도형 주석을 통해 실습할 수 있습니다.

#### 3) Root Hunter 보조 학습
5 Whys 기반 인터랙티브 활동을 통해 문제의 원인을 탐색하고 사고를 확장할 수 있습니다.

#### 4) 시각화 기반 학습 지원
간트 차트, 손익분기점 시각화 등 단계별 이해를 돕는 시각 자료를 제공합니다.

### 표현 방식
- 카드형 UI 3~4개
- 간결한 제목 + 2~3줄 설명
- 실제 기능 중심 문구 사용

---

## 7.3 로드맵 및 학습 흐름 섹션

### 목적
서비스 전체 구조를 한눈에 이해하도록 돕습니다.

### 포함 요소
- 로드맵 이미지 또는 단계 흐름 다이어그램
- 10단계 창업 교육 흐름 표시
- 단계별 색상 구분 또는 번호 체계 적용
- 필요 시 간단한 설명 문구 추가

### 전달 메시지
이 서비스는 단편적인 자료 모음이 아니라, 학습자가 창업 과정을 순차적으로 이해하고 따라갈 수 있도록 설계된 구조라는 점을 강조합니다.

---

## 7.4 10단계 커리큘럼 소개 섹션

### 목적
실제 교육 콘텐츠의 범위를 구체적으로 보여줍니다.

### 권장 구성
각 단계별로 번호, 한글 제목, 영문 보조 제목을 정리합니다.

### 예시 구성
1. 팀 빌딩
2. 아이디어 발상 및 5 Whys
3. 시장 조사와 페르소나
4. 공감과 SCAMPER
5. MVP 설계
6. 비즈니스 모델 캔버스
7. 마케팅 전략 4P / 4C
8. 운영과 역할 분담
9. 재무와 손익분기점
10. 피칭과 발표

### UI 방식
- 카드형 그리드
- 세로 타임라인
- 아코디언

### 기획 포인트
이 영역은 서비스의 핵심 경쟁력 중 하나이므로, 단순 제목 나열보다는 “교육 흐름이 체계적”이라는 인상을 주어야 합니다.

---

## 7.5 단계별 상세 소개 섹션

### 목적
스크롤을 깊게 내리는 사용자에게 커리큘럼의 밀도와 실제성을 보여줍니다.

### 내용 방향
- 각 단계에서 무엇을 배우는지
- 어떤 활동이 포함되는지
- 어떤 결과물을 만들게 되는지

### 활용 방식
이 섹션은 메인 랜딩에 전부 노출할 수도 있고, 길이가 길어질 경우 별도 소개 페이지 또는 “커리큘럼 더보기” 연결 구조로 분리할 수 있습니다.

### 권장 원칙
- 랜딩이 과도하게 길어지지 않도록 요약 중심으로 구성
- 상세 내용은 필요 시 확장 UI 사용

---

## 7.6 추천 학습 / 빠른 시작 섹션

### 목적
방문자가 바로 행동할 수 있도록 구체적인 시작점을 제시합니다.

### 구성 예시
- 1단계 워크북 시작하기
- Root Hunter 체험하기
- 추천 학습 3개 보기

### 기획 의도
“무엇을 할 수 있는지”를 이해한 뒤 바로 클릭할 수 있는 명확한 입구를 제공해야 합니다.

---

## 7.7 하단 CTA 섹션

### 목적
페이지를 끝까지 본 사용자를 실제 진입 행동으로 연결합니다.

### CTA 문구 예시
- 지금 Y-START 창업 여정을 시작해보세요
- 10단계 워크북으로 창업 아이디어를 구체화해보세요
- 첫 단계부터 차근차근 시작하세요

### 버튼 예시
- 1단계부터 시작하기
- 워크북 열기
- 전체 커리큘럼 보기

## 8. 콘텐츠 작성 원칙

### 8.1 사실 기반 원칙
랜딩 페이지의 모든 설명은 현재 서비스에 실제 존재하는 기능과 콘텐츠만 기준으로 작성합니다.

### 8.2 표현 원칙
- 교육 서비스답게 명확하고 신뢰감 있는 문장 사용
- 과도한 광고성 표현 지양
- 기능 설명은 구체적으로 작성
- 사용자가 바로 이해할 수 있는 단문 중심 구성

### 8.3 금지 원칙
- 구현되지 않은 AI 기능, 자동 분석 기능 등을 임의로 추가하지 않음
- 모호한 추상 표현만으로 구성하지 않음
- 서비스 실체와 다른 과장 표현 사용 금지

- 단계 구분이 명확한 색상 시스템 적용

### 9.3 공통 요소

- 모바일 대응 구조 필수
- 다크 모드 사용 시 가독성 유지

## 11. 랜딩 페이지 구현 방식 제안

### 방식 A. 현재 홈을 랜딩 페이지로 정제
기존 홈 구조를 유지하면서 카피와 정보 배열을 마케팅형 랜딩 구조로 정비하는 방식입니다.

**장점**
- 빠르게 구현 가능
- 기존 컴포넌트 재사용 가능
- 개발 비용 절감 가능

### 방식 B. 별도 랜딩 페이지 분리
루트 페이지를 랜딩 전용으로 구성하고, 실제 학습 대시보드는 별도 경로로 분리하는 방식입니다.

**장점**
- SEO 및 대외 홍보 최적화 가능
- 정보 구조를 보다 명확하게 설계 가능
- 서비스 소개와 실제 앱 흐름을 분리 가능

### 권장 방향
초기에는 **현재 홈을 랜딩 페이지 형태로 정제하는 방식**이 가장 효율적입니다.  
이후 서비스 확장 또는 외부 홍보 강화가 필요할 때 별도 랜딩 구조로 분리하는 것이 적절합니다.

## 13. 최종 정리

Y-START 랜딩 페이지는 단순 홍보 페이지가 아니라, **청소년 창업 교육용 디지털 워크북 플랫폼의 구조와 학습 가치를 설득력 있게 전달하는 진입 페이지**로 설계해야 합니다.  
따라서 카피 중심의 마케팅 페이지보다, 실제 기능과 커리큘럼이 명확히 보이는 구조가 더 적합합니다.

핵심은 다음 세 가지입니다.

- **10단계 창업 교육 구조가 명확해야 합니다.**
- **실제 학습 기능이 존재하는 서비스라는 점이 보여야 합니다.**
