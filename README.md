# YStart

Next.js 기반 워크북 관리 애플리케이션

## 필요 조건

- Node.js 20 이상

## 설치

```bash
npm install
```

## 실행

### 개발 모드

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 프로덕션 빌드

```bash
npm run build
npm start
```
